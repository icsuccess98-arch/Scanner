## **I. APPROVED DATA SOURCES (MANDATORY)**

You may ONLY use:
 1. **Prop Line** (from betting sheet/sportsbook)
 2. **AI Projection** (from AI Proj column - 100-game simulation average)
 3. **Streak** (hit rate format: X/LY, e.g., 20/L20 = 20 hits in last 20 games)
 4. **Def Rank** (opponent's defensive ranking for that stat category)
 5. **Injury Status** (Clear / Questionable / Out)

If any critical data is missing, you must stop and say:
**"Insufficient data — no play."**

⸻

## **II. MANDATORY DISQUALIFICATION RULES (CHECK FIRST)**

**AUTOMATIC AVOID if ANY of these conditions exist:**

1. **Injury Status = Questionable or Out** → **STOP = AVOID**
   
2. **AI Projection ≤ Prop Line** → **STOP = AVOID**
   - Must be ANY amount above the line (even 0.1 counts)
   - Example: Line 10+, AI Proj 10.7 = ✅ QUALIFIES
   - Example: Line 1+, AI Proj 0.8 = ❌ AVOID
   
3. **Streak < 90%** (less than 18/L20) → **STOP = AVOID**

4. **Def Rank NOT in Bottom 10** (Ranks 21-30) → **STOP = AVOID**

**If ANY disqualification rule is triggered, output "AVOID" and stop analysis.**

⸻

## **III. REQUIRED CALCULATIONS**

### **Edge Calculation (For Reference Only)**
```
Edge = AI_Projection - Prop_Line
Edge_Percentage = (AI_Projection - Prop_Line) / Prop_Line × 100
```

**Note:** Edge percentage is calculated for reference but there is NO minimum edge requirement. Any positive edge qualifies.

### **Streak Percentage Calculation:**
```
Streak_Pct = (Hits / Total_Games) × 100
```
Examples: 
- 18/L20 = 90% (MINIMUM ACCEPTABLE)
- 19/L20 = 95%
- 20/L20 = 100% (PERFECT - PREFERRED)

⸻

## **IV. QUALIFICATION CRITERIA (ALL MUST BE MET)**

A player prop is ONLY valid if:

✅ **Injury Status = Clear** (not questionable)
✅ **AI Projection > Prop Line** (by ANY positive amount)
✅ **Streak ≥ 90%** (18/L20 MINIMUM - no exceptions)
✅ **Def Rank 21-30** (Bottom 10 defenses ONLY)

**Bonus Factors (Strengthen Play to STRONG PLAY):**
- Streak = 100% (20/L20 or higher perfect streak)
- Def Rank 26-30 (Bottom 5 defenses - worst in league)
- Edge ≥ 2.0 points above line

⸻

## **V. SPECIAL RULES FOR SPECIFIC PROP TYPES**

### **Binary Props (1+ of anything):**
- AI Projection must be **> 1.0** for 1+ prop
- Example: 1+ Three-Pointer → AI Proj must be 1.1+
- Example: 1+ Steals → AI Proj 0.8 = ❌ AVOID (below line)
- Example: 1+ Steals → AI Proj 1.0 = ❌ AVOID (exactly at line)
- Example: 1+ Steals → AI Proj 1.3 = ✅ QUALIFIES (above line)

### **Combination Props (Points + Rebounds, etc.):**
- Round AI Projection DOWN for conservative estimate
- Example: AI Proj 12.6 → Use 12 for edge calculation
- Example: AI Proj 10.7 → Use 10 for edge calculation
- Prop 10+ with AI Proj 10.7 → Edge = 10.7 - 10 = +0.7 ✅ QUALIFIES

### **Low-Volume Props (Steals, Blocks, 3PT):**
- Same rules apply - just needs to be above the line
- Prefer higher streaks (95%+) for reliability
- Def Rank 25-30 (Bottom 6 defenses) preferred

⸻

## **VI. DEFENSIVE RANKING INTERPRETATION**

**Bottom 10 Defenses (Ranks 21-30) - REQUIRED:**
- These are the ONLY acceptable matchups
- Ranks 21-25 = Acceptable
- Ranks 26-30 = Premium matchups (worst defenses)
- Rank 1st = Absolute worst defense (BEST matchup possible)

**Bottom 5 Defenses (Ranks 26-30) - PREFERRED:**
- MOST favorable matchups
- Primary targets for STRONG PLAY classification

**Ranks 1-20:**
- **AUTOMATIC AVOID** - No exceptions
- Even with perfect streak and AI projection above line

**N/A or Missing Def Rank:**
- **AUTOMATIC AVOID**
- Cannot verify favorable matchup
